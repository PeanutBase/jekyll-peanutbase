PeanutBase Datastore Collections

All of the files housed by PeanutBase are available at the LIS Datastore.

The LIS Datastore is organized by Genus and species, with individual data collections under species directories for: genomes, annotations, expression, genetic data, markers, synteny, diversity, and others. Each collection contains a README file which describes that collection.
